Loads the scene based on the config at config path and scene files at scene path.

It uses tinyobj library to parse the obj and mtl files and then converts the geometry to the custom classes It uses TBRDF library to load BRDF materials It then builds the BVH tree and converts it to the compacted version BVH tree construction may be parallelized using openmp if toggled on in context

Parameters
[in]config_pathpath to the config file
[in]scene_pathpath to the scene directory
[in]scene_namename of the scene in the directory
